# GLOSSARY.md — ResearchPilot

*   **ADK**: Agent Development Kit (Google's agent orchestration SDK).
*   **Semantic Scholar**: A free, AI-backed search engine for academic publications.
*   **ArXiv**: An open-access archive for scientific papers in physics, computer science, and related fields.
*   **Critic Agent**: The LLM-as-judge validator checking research synthesis factual alignment.
*   **Supervisor Agent**: The main coordinator managing task decomposition and agent routing.

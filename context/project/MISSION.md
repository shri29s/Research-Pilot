# MISSION.md — ResearchPilot

## Mission
Accelerate academic research onboarding by automating the literature review process, providing a cited, synthesized, and quality-audited literature draft in minutes instead of weeks.

## Success Criteria
1. **Accuracy**: 100% citation truthfulness — no hallucinated papers in the final output.
2. **Efficiency**: Complete a multi-paper search, vectorization, synthesis, and review loop in under 3 minutes.
3. **Observability**: Logs every API call, latency metric, and LLM-as-judge score for auditability.
4. **Resiliency**: Robust retry mechanisms for Semantic Scholar API rate limits.
